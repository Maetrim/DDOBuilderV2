Enhancements Pane

The Enhancements pane display can be toggled by clicking the button in the Standard Toolbar.

This view allows you to select and train enhancements by spending Racial, Universal and regular Action Points.

The number of available Racial, Universal and regular Action Points are shown in the windows title bar.

You can gain access to the Universal enhancement trees by toggling the tree select buttons under the racial enhancement tree.

All characters can have a maximum of 7 enhancement trees. One Racial tree, and 6 Class/Universal trees. If the tree you want is not one of the 7 you can choose it using the drop list combo boxes displayed under the Class/General tree options:

Selecting a new tree will switch out the current tree selection. The current tree can only be switched out if no Action Points have been spent in it.

The Class and Universal trees can be re-ordered using drag and drop of the tree icon. To switch the order click and drag a tree icon onto another tree. Those trees will be swapped.

Any given enhancement tree can have the following actions done:

  • Left click a highlighted Enhancement to train that enhancement. Can only be done on enhancements with a white highlight border
  • Right click an item with the Revoke icon to revoke that enhancement in this tree. Items cannot be revoked if they are pre-requisite of other items such as required points spent or a direct requirement.
  • Mouse over an enhancement to get a description of its effects.
  • You can left click an option with a multi-selector to view those options in the selection dialog, even if you cannot train the enhancement.
  • Reset all Action Points spent in this tree.
  • Development Option: A Middle click on the tree will copy the displayed bitmap to the clipboard.
  • Development Option: A Middle click on an item with the SHIFT button on the keyboard down will allow the Icon transparency to be updated with a number of mask options. (Action can be cancelled)
  • Development Option: A Left click on an item with the SHIFT and CONTROL buttons on the keyboard also held down will allow an Arrow item and the required enhancement requirement to be added to the tree data file. (Action can be cancelled)

Individual tree enhancements can have:

Multiple Ranks, each to be bought separately.

Additional requirements, such as required trained Feats to be selected.

A multiple select option dialog when choosing a variant of a specific enhancement

Options in the multi-selector dialog may be disabled if you do not meet the individual requirements for them.

Multi-selectors for item not yet trainable can also be reviewed. The dialog will be displayed with all items disabled.
