Skills Pane

The Skills pane display can be toggled by clicking the button in the Standard Toolbar.

This pane allows you to set your Skill Point spends for all your Heroic levels at the same time. Move your mouse over the view to select the level and Skill to train. This is shown by selection lines around the Skill and Level.

  • To train a skill rank at that level left click that location, to revoke a skill rank, right click that location.
  • Class skills are shown with a Green background.
  • Cross Class skills are shown with a White background.
  • Untrainable skills are shown with a Red background (Typically Disable Device and Open Lock if you do not have a trapping class.
  • An overspend in a skill is shown with the numbers appearing in Red.
  • The Tome column allows you to set the Skill Tome you have for that given skill.
  • Clicking a column header will highlight that column.
  • Left clicking a Skill name will try and spend the maximum number of skill points (if available) for that skill.
  • Right clicking a Skill name will revoke all skill points spent in that skill.
  • Left clicking the word Tome under the top symbol will add 1 to all skill tomes is possible.
  • Right clicking the word Tome under the top symbol will subtract 1 to all skill tomes if possible.
  • Auto Buy will spend all available skill points at all levels on class appropriate skills.
  • Clear Skills will clear all spent skill points.
  • The Final column shows the current Skill total (from the breakdowns) for all skills, with totalled values from all sources.
