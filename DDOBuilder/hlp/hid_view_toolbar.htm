Standard Toolbar

See also Menus | File Menu | Build Menu | Edit Menu | Forum Export Menu | Development Menu | Gear Menu | Settings Menu | View Menu | Help Menu

The standard toolbar can be toggled from the View | Toolbars and Docking Windows | Standard menu command.

When an item is grey it means that the action is currently unavailable.
