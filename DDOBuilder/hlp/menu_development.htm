The Development menu offers the following commands:

  Menu Command Description
  Dump weapon groups to log Dumps internal Weapon Groups to the Log window to see what state they are currently in.
  Show 0 value breakdown entries Forces the Breakdown view to show entires that evaluate to a 0 value to check on stacking issues.
  Run Wiki Item Crawler Downloads and parses item pages from the DDO Wiki, for the auto generation of .item files at a later time.
  Crawl Specific Wiki Link Downloads and parses a specific DDO Wiki page, for the auto generation of .item files at a later time.
  Process Wiki Files Processes downloaded Wiki item pages and auto generates .item files for them.
  Process Specific Wiki File Allows you to select a specific downloaded Wiki file and process its content to create a .item file.
  Update Accessory Item Images Allows you to associate images with auto generated .item files for all types of Accessory items.
  Update Weapon Item Images Allows you to associate images with auto generated .item files for all types of Weapons.
  Verify Loaded Data Looks at all loaded data for detectable data errors. Reports any to the Log window.

Note that the development menu commands are used by those who update and maintain the DDOBuilder software.
