File Menu

See also Menus | Edit Menu | Development Menu | Gear Menu | Settings Menu | View Menu | Help Menu | Standard Toolbar

The File menu offers the following commands:

  Menu Command Description
New Creates a new empty document. Will prompt to save current changes, if any.
Open... Opens an existing .DDOBuild saved document.
Save Saves any changes to the current .DDOBuild file.
  Save As... Allows you to save the current .DDOBuild to a new location/filename.
  Import V1 file... Imports an existing DDOBuilder V1 .ddocp file into the current document.
  Recent File List Open a dynamic reference to a recently opened .DDOBuild file.
  Exit Exits DDOBuilder, prompts to save any changes to the current .DDOBuild file, if any.
