The Gear menu offers the following commands:

  Menu Command Description
New... Creates a new empty Gear set for the selected build
Copy Copies the currently selected Gear set to the clipboard.
Paste Creates a new named Gear set using the content of the clipboard.
Delete Removes the current Gear set from the build.
Import .gearset file... Allows you to Import a .gearset file created from the DDO Named Gear Planner application.
Import gear set from clipboard... Allows you to Import a Gear set from the clipboard created by the Gear Planner website.
