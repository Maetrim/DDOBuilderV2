Settings Menu

The Settings menu offers the following commands:

  Menu Command Description
Show only Epic feats for Epic feat slots When enabled filters "Heroic" feats from the list of selections for Epic and Legendary Feat selections
Show Unavailable Feats When enabled, the Feat selection drop lists will show all feats, regardless of whether you qualify for their selection. This option can aid you in finding out why you do not quality for a feat selection.
Ignore Lists Active When enabled, any Feats or Spells in your custom Ignore list will not show up in the selection interface.
Lamannia Mode When enabled, any builder features that have been implemented for game feature under test on Lamannia will be available.
Auto Select Single Option Enhancements When enabled, if you try and select an enhancement which has options, but only one selection would be available, then that selection will be automatically made without showing the selection dialog.
